CORE ELEMENTS

Everyting in this website was designed behind the idea of simple grid, where different areas of the page held different content. At the top of my grid I have a horizontal navigation bar and at the bottom I have a horizontal social media bar. Becuase every page on my website maintained these bars I only had to create them once and paste them on every html page. You can view the code I used for these two elemnts on my code pen above. Within the middle of my page I organized my content using a CSS property called flexbox which allowed me to create adaptive areas of information based on the size of the browser.
